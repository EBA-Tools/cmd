Commands

  • Admin - Restart EBA Command Center and run it as administrator.
  • Backup - Create a backup of EBA Command Center data.
  • Config - Edit EBA Command Center settings. Login required. This command has sub-commands:
    • Account - Edit Accounts. This command has sub-commands:
      • New - Create a new account.
      • Edit - Modify an existing account.
    • Cmd - Edit Commands. This command has sub-commands:
      • New - Create a new command.
      • Edit - Modify an existing command.
    • More commands exist, but are currently not listed.
  • Crash - Crashes EBA Command Center. Debugging command.
  • Dev - Enable developer mode. Command does not exist in the latest version.
  • End - Exit EBA Command Center. Command will also run if the command box is empty, or the Cancel button is pressed.
  • Error - Disable Error Handle. Replaced by a Modifiable Startup Setting in the latest version.
  • Export - Export a command.
  • Help - Open the EBA Command Center help page.
  • Import - Import an exported command, a backup file, a plugin file, or a startup key.
  • Login - Login to your EBA Cmd Account.
  • Logout - Log out of your EBA Cmd Account.
  • Logs - Open the EBA Command Center log file.
  • Migrate - Begin the Migration process.
  • Plugin - View the currently loaded plugins.
  • Read - Display text from a text file.
  • Refresh - Reinstall EBA Command Center. Login required.
  • Restart - Restart EBA Command Center.
  • Run - Execute a file, open a folder, or run a command in Command Prompt. Replaced by a command with parameters.
  • Run {1} - [Includes Parameter] Execute a file, open a folder, or run a command in Command Prompt.
  • Shutdown - Open the EBA Command Center shutdown box.
  • Uninstall - Uninstall EBA Command Center. Login required.
  • Update - Visit the EBA Command Center website to download the latest version.
  • Upgrade - Upgrade your edition of EBA Command Center. This command no longer exists.
  • Ver - Get version information.
  • Version - Shortcut to the command 'ver'.
  • Write - Write data to a text file, or create a new text file.
